# Editorial AI Assistant

A fine-tuned language model for professional editorial assistance.

## Project Structure
- `scripts/` - Training and inference scripts
- `data/` - Training dataset (CSV format)
- Models are not included due to size (>15GB)

## Setup
1. Install dependencies: `pip install torch transformers peft datasets`
2. Prepare data: `python scripts/prepare_data.py`
3. Train model: `python scripts/train_model.py`

## Trained Models
Due to size constraints, trained models are not included in this repository.
Contact the maintainer for access to pre-trained models.
